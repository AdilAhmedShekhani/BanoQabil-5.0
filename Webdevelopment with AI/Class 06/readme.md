## 🎯 Flexbox (Easy Layout System)

Flexbox helps us arrange items in a row or column.

### 🔹 Basic Example:
```html
<div style="display: flex;">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>
```

👉 Important properties:
- display: flex  
- justify-content  
- align-items

## 📦 CSS Grid (Basic Idea)

Grid is used for **2D layout (rows + columns)**

### 🔹 Example:
```html
<div style="display: grid; grid-template-columns: 1fr 1fr;">
  <div>Item 1</div>
  <div>Item 2</div>
</div>
```
