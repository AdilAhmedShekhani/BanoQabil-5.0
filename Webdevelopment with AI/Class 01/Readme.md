## 📄 HTML Basics

We learned the foundation of HTML:

### 🔹 Boilerplate Structure

Basic HTML structure looks like this:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body></body>
</html>
```

---

### 🔹 Tags (Very Important)

- HTML works using **tags**
- Every tag has:
  - Opening tag → `<tag>`
  - Closing tag → `</tag>`

Example:

```html
<h1>Hello World</h1>
```

---

### 🔹 Headings

We learned heading tags from **h1 to h6**:

```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
```
