## 🤖 Prompt Engineering Basics

### 🔹 What is Prompt Engineering?
Prompt engineering means:
👉 Asking AI in a **clear and smart way** to get better results.

---

### 🔹 Writing Effective Prompts

Good prompt example:
- "Create a simple HTML page with headings and paragraph"

Bad prompt example:
- "HTML code do"

👉 Clear prompts = Better results ✅

---

### 🔹 Prompt Patterns

We learned different ways to use AI:

- **Question Asking**  
  → Ask AI to explain concepts  

- **Code Generation**  
  → Ask AI to create HTML/CSS code  

- **Debugging Help**  
  → Ask AI to fix errors in code  

---

## 🌐 HTML Elements (Using AI)

We used AI to generate and understand different HTML elements:

### 🔹 Headings
```html
<h1>Main Heading</h1>
<h2>Sub Heading</h2>
```

### 🔹 Paragraph
```html
<p>This is a paragraph</p>
```

### 🔹 Lists
```html
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
```

### 🔹 Links
```html
<a href="https://www.google.com">Visit Google</a>
```

### 🔹 Images
```html
<img src="image.jpg" alt="Sample Image">
```

### 🔹 Tables
```html
<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Ali</td>
    <td>20</td>
  </tr>
</table>
```
