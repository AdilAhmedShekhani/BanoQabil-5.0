## 🎨 CSS Basics

CSS is used to **style HTML elements**.

---

## 🔹 Types of CSS

### 1. Inline CSS
Applied directly inside the tag

```html
<h1 style="color: red;">Hello World</h1>
```

---

### 2. Internal CSS
Written inside `<style>` tag in `<head>`

```html
<style>
h1 {
  color: blue;
}
</style>
```

---

### 3. External CSS
Written in a separate `.css` file

```html
<link rel="stylesheet" href="style.css">
```

---
