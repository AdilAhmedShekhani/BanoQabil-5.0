## 📄 Basic HTML Structure

We learned the standard HTML structure:

```html
<!DOCTYPE html>
<html>
<head>
    <title>My Page</title>
</head>
<body>

</body>
</html>
```

---

## 🔹 Headings

HTML provides 6 heading tags:

```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
```

👉 `<h1>` is the biggest and `<h6>` is the smallest.

---

## 🔹 Paragraph Tag

We used paragraph tag to write text:

```html
<p>This is a paragraph</p>
```

---

## 🔹 Text Formatting Tags

We learned how to style text using:

```html
<b>Bold Text</b>
<i>Italic Text</i>
<u>Underlined Text</u>
```
